a tiny, always‑ready place for text.

micronote is a lightweight windows utility designed for quick notes, scratch text, and temporary thoughts. it stays out of the way, remembers everything automatically, and fades into the background until you need it again.

🏛️ built with Delphi 🏛️.  
no ads, no telemetry, no nonsense.

---

## features

- **autosave**  
  your text is saved every few seconds with zero interruption.

- **drag & drop text**  
  drop text from browsers, editors, or other apps directly into the note.

- **pin modes**  
  - **embed** — stays behind everything  
  - **float** — stays on top  
  - **free** — behaves like a normal window  

- **opacity fade**  
  micronote gently dims when not hovered, and returns to full opacity when you move your mouse over it.

- **dark + light themes**  
  clean, modern windows 11 styles with automatic icon switching.

- **global hotkey toggle**  
  show or hide micronote instantly using scroll lock, pause, or insert.

- **autotrim**  
  optional whitespace cleanup for tidy notes.

- **status bar indicators**  
  live word count, line count, and quick access to settings.

---

## why micronote?

micronote is intentionally small. it’s not a full note‑taking system, not a markdown editor, not a database. it’s a scratchpad;
 the digital equivalent of a sticky note that’s always there, always saved, and never in the way.

if you need a place to paste something, jot something down, or keep a temporary list, micronote is built for that.

## installation

download the latest release from the releases page and run `micronote.exe`.  
no installer, no setup — it just works.

---

## settings

micronote stores its configuration in:

\`\`\`
%userprofile%\\documents\\micronote.ini
\`\`\`

your text is stored in:

\`\`\`
%userprofile%\\documents\\micronote.txt
\`\`\`

both files are human‑readable and portable.
